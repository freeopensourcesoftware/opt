Windows Settings Tweaks
├── [Gaming Mode]
│   ├── Turn on gaming mode (search "turn on game mode")
│   ├── In game mode settings click on graphics
│   └── In Graphics click "change default graphics settings"
│       ├── Turn on "Optimizations for windowed games"
│       └── If available, turn on "Hardware-accelerated GPU scheduling"
├── [Security]
│   └── Search "Windows Security" and in "Device security" turn off "Memory integrity"
├── [Power Settings]
│   ├── Open terminal/powershell and run: powercfg -duplicatescheme e9a42b02-d5df-448d-aa00-03f14749eb61
│   └── Search "Edit Power Plan" and select "Ultimate Performance" from dropdown under "Hide additional plans"
├── [Personalization]
│   └── Open Settings > Personalization > Dynamic Lighting
│       ├── Turn off "Use Dynamic Lighting on my devices"
│       └── Turn off "Compatible apps in the foreground always control lighting"
├── [Apps]
│   ├── Open Settings > Apps > Advanced app settings
│   │   └── Set "Share across devices" to "Off"
│   └── (Optional) Open Settings > Apps > Startup
│       └── Disable everything you don't need
├── [Accessibility]
│   └── Open Settings > Accessibility > Visual effects
│       ├── Turn off "Animation effects"
│       └── Turn off "Transparency effects"
├── [Privacy & Security]
│   └── Open Settings > Privacy & security > Windows permissions
│       └── Disable all permissions
├── [Windows Update]
│   └── Open Settings > Windows Update > Advanced options
│       ├── Disable the top 4 settings
│       └── Click "Delivery Optimization" and turn off "Allow downloads from other devices"
├── [System Performance]
│   └── Search "View advanced system settings" > Advanced tab > Performance > Settings
│       ├── Select "Adjust for best performance"
│       ├── Enable "Smooth edges of screen fonts"
│       ├── Enable "Show thumbnails instead of icons"
│       └── Enable "Show window contents while dragging"
├── [Registry Tweaks]
│   ├── Open Regedit (search "regedit")
│   ├── Navigate to HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Multimedia\SystemProfile\Tasks\Games
│   │   ├── Set GPU Priority to 8
│   │   ├── Set Priority to 6
│   │   ├── Set Scheduling Category to "High"
│   │   └── Set SFIO Priority to "High"
│   ├── Navigate to HKLM\SYSTEM\CurrentControlSet\Control\PriorityControl
│   │   └── Set Win32PrioritySeparation to 38
│   └── Navigate to HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Multimedia\SystemProfile
│       ├── Set NetworkThrottlingIndex to ffffffff (Decimal) - create if not exists
│       └── Set SystemResponsiveness to 0
│   ├── Navigate to HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\DriverSearching
│   │   └── Set SearchOrderConfig to 0
│   ├── Navigate to HKLM\SYSTEM\CurrentControlSet\Control\Session Manager\Power
│   │   └── Set HiberbootEnabled to 0
│   ├── Navigate to HKLM\SOFTWARE\Microsoft\PolicyManager\default\ApplicationManagement\AllowGameDVR
│   │   └── Set value to 0 - only if you don't use GameBar
│   ├── Navigate to HKLM\SYSTEM\CurrentControlSet\Control\Power
│   │   └── Set HibernateEnabled to 0
│   ├── Navigate to HKCU\Control Panel\Desktop
│   │   └── Set MenuShowDelay to 0
│   ├── Navigate to HKLM\SYSTEM\CurrentControlSet\Control
│   │   └── Set WaitToKillServiceTimeout to 2000
│   └── Navigate to HKLM\SYSTEM\CurrentControlSet\Control\Power\PowerSettings\54533251-82be-4824-96c1-47b60b740d00\943c8cb6-6f93-4227-ad87-e9a3feec08d1
│       └── Set Attributes to 3
└── [Optional]
    └── Search "Notifications & actions" and turn off "Notifications"
        └── Note: The notifications service uses a lot of resources
